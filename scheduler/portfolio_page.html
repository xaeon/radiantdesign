Расписание

Данное приложение является составной частью информационной системы для учебных заведений.

Отображение расписания является основной функцией приложения. Расписание может выводится в разных режимах и адаптивно подстраиваться под размер экрана. Также есть возможность задавать фильтры по различным параметрам для уточнения выборки. У пользователя также есть возможность осуществлять поиск по мероприятиям.

Также есть возможность прикрепления текстовых заметок и файлов к мероприятиям с возможностью их дальнейшего распространения. Реализованы функции хранения и управления загруженными файлами аналогично таким сервисам как Google Drive и Яндекс Диск.
