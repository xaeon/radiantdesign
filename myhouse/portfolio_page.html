Мой Дом

Приложение для сервиса Мой Дом.

С помощью этого приложения пользователь может получить оперативную информацию с точностью до определенного дома. Приложение информирует пользователя об отключениях, авариях и времени их устранения а также других важных новостях.

Важным разделом являются новости города. Там пользователь может узнать про актуальные события своего города. Аналогичным функционалом обладает раздел администрации города.
